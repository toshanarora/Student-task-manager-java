# Student Task Manager (Java)

## Overview
The Student Task Manager is a simple Java-based application designed to help students manage their daily tasks and academic responsibilities efficiently. It allows users to add, track, and manage tasks with deadlines using a command-line interface.

---

## Features
- Add new tasks with deadlines  
- View all tasks  
- Mark tasks as completed  
- Delete tasks  
- Persistent storage using file handling  

---

## Technologies Used
- Java  
- Object-Oriented Programming (OOP)  
- File Handling  
- ArrayList (Collections Framework)  

---

## How to Run

1. Open terminal in the project folder  
2. Compile the program:
```
javac *.java
```

3. Run the application:
```
java Main
```

---

## Usage
- Choose options from the menu displayed in the terminal  
- Enter task details when prompted  
- Manage tasks using the available options  

---

## Problem Statement
Students often forget assignments and deadlines due to poor task management. This project provides a simple and effective solution to organize and track tasks.
